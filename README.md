# ESP32-S3 BB-8 Droid 🤖

A fully DIY BB-8 inspired robot built using ESP32-S3 with Wi-Fi based real-time control.

This project combines embedded systems, robotics, IoT, and mechanical design into a working mobile robot.

---

## 🚀 Key Features

- ESP32-S3 microcontroller
- Wi-Fi Access Point mode (no router required)
- Web-based control interface (runs in browser)
- Real-time motor control
- Fully custom 3D printed mechanical structure
- DIY electronics + firmware integration

---

## 📡 How to Use

1. Power on the robot  
2. Connect to Wi-Fi network: **BB-8**  
3. Enter password: **123456789**  
4. Open browser and go to:  
   ```
   192.168.4.1
   ```
5. Use the control panel to move the robot

---

## 🧠 System Architecture

- ESP32-S3 runs as Wi-Fi Access Point
- Embedded web server serves control UI
- Browser sends commands via HTTP requests
- ESP32 processes commands and drives motors

---

## 🛠 Hardware Used

- ESP32-S3
- Motor Driver Module
- DC Motors
- Battery System
- Custom 3D Printed Body (BB-8 inspired)
